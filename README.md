# 🏠 House Price Prediction

This project predicts house prices using the **Random Forest Regression** algorithm.

## Features
- Load Dataset
- Label Encoding
- Train-Test Split
- Random Forest Regression
- Model Evaluation
- Save & Load Model

## Technologies Used
- Python
- Pandas
- NumPy
- Scikit-learn
- Joblib

## Files
- `house_price_model.ipynb` – Model training notebook
- `house.csv` – Dataset
- `house_price_model.pkl` – Trained Model
- `app.py` – Streamlit web application
